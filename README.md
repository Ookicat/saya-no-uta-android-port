# Saya no Uta Android Edition

A fan-made Android port of the classic visual novel **Saya no Uta (The Song of Saya)**. This project aims to bring the experience to mobile devices using the Flutter framework, based on the previous DS port.

## Credits

This project is built upon the hard work of the community:

- **Translation:** Makoto
- **Original DS Port & Assets:** Anonymous

*Note: Credits derived from `assets/script/main.scr` (Saya no Uta DS Edition v1.0).*

## Installation
You can download the latest version of the game from the **[GitHub Releases page](https://github.com/Ookicat/saya-no-uta-android-port/releases)**.
